# 🎮 HTML Games Collection - Android APK

**30 Fun HTML/CSS/JavaScript Games in a Single Android App**

## 📱 About This APK

This Android application contains all 30 HTML/CSS/JavaScript games in a single mobile app. The app uses WebView to display the games, making them fully playable on Android devices.

## 🎯 Features

- ✅ **30 Complete Games** - All games from the web version
- ✅ **Mobile Optimized** - Touch controls and responsive design
- ✅ **Offline Play** - All games work without internet
- ✅ **Fast Loading** - WebView optimized for performance
- ✅ **Easy Navigation** - Simple interface to browse games

## 🚀 How to Install APK

1. **Download** the APK file from GitHub Releases
2. **Enable Unknown Sources** in Android Settings
3. **Install** the APK file
4. **Open** the app and enjoy playing!

## 🛠️ Technical Details

- **Framework**: Cordova/PhoneGap
- **Platform**: Android 5.0+ (API 21+)
- **WebView**: Modern WebView with JavaScript enabled
- **Storage**: No data storage required
- **Permissions**: Internet access for WebView

## 🔧 Build Instructions

If you want to build the APK yourself:

### Prerequisites
- Node.js installed
- Cordova CLI: `npm install -g cordova`
- Java JDK 11+
- Android SDK with API 21+

### Build Steps
```bash
cd android-app
cordova build android
```

The APK will be generated in: `platforms/android/app/build/outputs/apk/debug/`
